# STAAR Prep App - 5th Grade

A web-based practice tool designed to help 5th graders prepare for the Texas STAAR exam. The app features:
- Interactive quizzes for multiple subjects.
- Unlimited algorithmic math practice questions.
- Instant feedback and score tracking.
- Simple, kid-friendly interface.

## How to Deploy (Make it live!)

To make this app visible on the internet (github.io), you need to enable GitHub Pages:

1.  Go to your repository on GitHub: [https://github.com/stsaeed/staar-prep-app](https://github.com/stsaeed/staar-prep-app)
2.  Click on the **Settings** tab (gear icon) at the top right of the repo page.
3.  In the left sidebar, click on **Pages**.
4.  Under **Build and deployment** > **Branch**, click the dropdown that says "None" and select **main**.
5.  Ensure the folder dropdown says **/(root)**.
6.  Click **Save**.

After a minute or two, your site will be live at:  
**https://stsaeed.github.io/staar-prep-app/**
